# Debug View

Inspect workflow execution tree.

## Usage

From dashboard: Press `Enter` on any workflow

CLI:
```bash
pumbaa workflow debug <workflow-id>
```

## Keys

| Key | Action |
|-----|--------|
| `↑/↓` or `k/j` | Navigate tree |
| `→/←` or `l/h` | Expand/collapse |
| `Enter` or `Space` | Toggle expand |
| `d` | Show call details |
| `o` | Show logs |
| `q` or `Esc` | Back |
